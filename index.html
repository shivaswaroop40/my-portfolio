Selected projects

ML-Driven Kubernetes NetworkPolicy Generation

Master thesis · eBPF · Cilium · Hubble · Kubernetes

Designed a data-driven system that learns baseline pod-to-pod communication from eBPF flow data (Cilium/Hubble) and automatically synthesizes Kubernetes NetworkPolicy objects to enforce least-privilege segmentation in a single-cluster environment. Implemented and evaluated the approach on real-world microservice benchmarks (Train Ticket, Sock Shop), using a supervised ML classifier over curated “known-good” traffic and avoiding iterative retraining to reduce the risk of adversarial poisoning.

Ankra AB · Master thesis · Jan 2026 – Present

Kubernetes in a Datacenter Network

Configured hybrid edge/cloud setup with two Kubernetes clusters using VXLAN for secure overlay networking. Deployed Llama2 LLM as distributed microservices and implemented RDMA over fabric for ultra-low latency remote storage access.

KTH Royal Institute of Technology • January – March 2025

Network Systems Security Implementation

Built enterprise-grade secure networked system with complete PKI implementation using XCA and OpenSSL. Configured certificate-based authentication, advanced IPTables firewall rules, and hardened NextCloud deployments with SSL/TLS.

KTH Royal Institute of Technology • January – March 2025

Email Security with GPG

Implemented comprehensive email encryption system using GNU Privacy Guard on Linux. Built end-to-end encryption with digital signatures and key management infrastructure for secure communication.

KTH Royal Institute of Technology • November 2024 – January 2025

ISP Implementation

Linux · Quagga · Kathara

Implemented a small-scale software-based Internet Service Provider model on Linux using Kathara, a network emulation platform. Configured an Autonomous System with OSPF for internal routing and BGP for external connectivity between autonomous systems.

KTH Royal Institute of Technology · Course: IK2215 Advanced Internetworking · Aug 2024 – Oct 2024
